# FYPReactApp

A cross-platform mobile application built using **React Native**, styled with **NativeWind** (Tailwind CSS for React Native), and powered by the **Gluestack UI** component library.

## 🛠️ Tech Stack

- **React Native** – Build native mobile apps using JavaScript and React.
- **Expo** – Simplifies React Native development and testing.
- **NativeWind** – Tailwind CSS-style utility classes for React Native.
- **Gluestack UI** – Accessible and customizable UI component library for React Native.
